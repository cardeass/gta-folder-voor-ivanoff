 重要
_______________________________


一回限り有効なロックスターアクティベーションコードによるインターネットでの登録が必要です。登録はロックスター・ゲームスSocial Clubアカウントをお持ちの方(13歳以上)に限られ、1つのロックスターアクティベーションコードにつき一回限り有効です。 
1つのSocial Clubアカウントにつき、1つのPCからのログインに常時限られます。使用済みのロックスターアクティベーションコードは譲渡できません。Social Clubアカウントは譲渡不可なものとします。
本ソフトウェアの利用は、ゲーム内およびrockstargames.com/eula/?locale=jpの利用規約に従っており、オンラインアカウント利用規約はja.socialclub.rockstargames.com/に記載されています。 
使用許諾契約、行動規範またはその他のポリシーを違反した場合、ゲームまたはオンラインアカウントへのアクセスを制限または停止させていただく場合があります。カスタマーサポートおよびテクニカルサポートは、support.rockstargames.com/hc/jaをご覧ください。 
マルチプレイサービスやボーナスコンテンツを含む各種サービスや機能、限定コンテンツ、アンロック要素、ダウンロード コンテンツ、オンライン コンテンツなど、特別な仕様に対する譲渡不能なアクセスには、一回限り有効なロックスターコード、追加費用、オンラインのアカウント登録(13歳以上)が必要になる場合があります。 
特別な仕様に対するアクセスにはインターネット接続環境が必要となる場合や、すべてのユーザーがご利用いただけない場合があります。また、通知から30日が経過後、これらを停止、変更、もしくは規約を変更して提供する場合があります。 
許諾を得ずに、このソフトウェアの複製、リバースエンジニアリング、配信、公共の場における実演、レンタル、商業目的での使用、コピープロテクトの解除を行うことは固く禁じられています。

本製品をインストールして起動するには、管理者権限が必要です。また、本製品に使用するユーザーアカウントに全面的な管理者権限が与えられている事をご確認ください。

2) インストール方法
_____________________________

ステップ:
	
1.  (ディスク版の場合)『グランド・セフト・オートV』(Disc 1)をDVD-ROMドライブに挿入します。
インストーラーが起動しない場合は、DVD-ROMドライブのフォルダを開き、「セットアップ」アイコンをダブルクリックしてください。

2.  『グランド・セフト・オートV』のインストールを開始します。画面の指示に従い、インストールを完了させてください。

3.  DirectX、Microsoft .NET、Visual C++など、その他のコンポーネントを必要に応じてインストールします。画面の指示に従ってください。

4.  DVD-ROMドライブにディスクを挿入せずに『グランド・セフト・オートV』をお楽しみいただけます。

5.  『グランド・セフト・オートV』のアイコンをクリックして『グランド・セフト・オートV』を開始します。アップデートが適用できる場合は、ダウンロードしてインストールを行います。

6.  Social Clubアカウントの作成、または既存アカウントへのサインインを求められます。

7.  ロックスターアクティベーションコードを入力し、インターネット接続によるワンタイム製品アクティベーションを行います。ロックスターアクティベーションコードは解説書の裏面または購入時に送信されるメール内に記載されています。

本製品にはリリース日チェック機能が搭載されているため、発売日までゲームを起動できません。接続がうまくいかない場合、または処理に失敗した場合は、support.rockstargames.com/hc/jaで詳細をご確認ください。ロックスターアクティベーションコードは再発行できないため、解説書は大切に保管してください。

3) パッチの適用
_____________________________

Steam版の場合は、Steamのパッチシステムが使用されます。

パッチはsupport.rockstargames.com/hc/jaからも手動でダウンロード出来ます。

6) PCコントローラー
_____________________________

PC版『グランド・セフト・オートV』は、マウス、キーボード、Xbox 360 Controller for Windows などのXinputコントローラーに対応しています。

全ての操作タイプは互換的に使用できます。例えば、徒歩時にマウスとキーボードで操作し、オプションを一切変更することなく、いつでも Xbox 360 コントローラーに切り替え可能です。

マウス、キーボード、Xinputコントローラーのゲーム内ヘルプは自動で更新され、前回使用した操作タイプの正しいキー/ボタンを表示します。

7) お問い合わせとサポート情報
___________________________________

技術的な問題が生じている場合、support.rockstargames.com/hc/jaでサポートに関する最新情報やよくある質問とその回答をご確認ください。 
ウェブサイト、Eメール、電話およびTwitterを通じてサポートさせていただきます。

注意: PC版タイトルに関するサポートをご希望の場合は、お問い合わせ時にDxDiagおよびシステム情報をご添付ください。 

この情報は、以下の方法で入手できます:

DxDiagレポートをエクスポートする

・[スタート]>[プログラムとファイルの検索]の欄へ、「dxdiag」と入力して[Enter]を押す（入力の際「」は付けないでください）
・緑のバーで表示されるロードが終了後、[情報をすべて保存...]を押す
・ファイルに分かりやすい名前（例：日付と氏名）を付け、デスクトップ上など見つけやすい場所へ保存する
・[終了]を押してこのアプリケーションを閉じる

MSInfoレポートをエクスポートする
・[スタート]>[プログラムとファイルの検索]の欄へ、「msinfo32」と入力して[Enter]を押す（入力の際「」は付けないでください）
・[システム情報]のウィンドウが開いたら、ツールバーの[ファイル]を押して、エクスポートを押す
・ファイルに分かりやすい名前（例：日付と氏名）を付け、デスクトップ上など見つけやすい場所へ保存する
・エクスポートには数分時間がかかる場合があります。バーが一杯になるのを待って、アプリケーションを閉じてください。

お問い合わせの際は、発生した問題の詳細と表示されたエラーメッセージのテキストをご用意ください。さらに、エラーメッセージのスクリーンショットと両方のテキストファイルを送付していただけますと、解決に役立ちます。

エラーメッセージのスクリーンショットを撮る場合
・エラーが出ているウィンドウを閉じないよう注意し、このウィンドウのタイトルバーをクリックする
・[ALT]と[PrintScrean]キーを同時に押す
・エラーが出ているウィンドウを閉じないよう注意し、[スタート]>[プログラムとファイルの検索]の欄へ、「mspaint」と入力して[Enter]を押す（入力の際「」は付けないでください）
・[ペイント]が開いたら、[Ctrl]と[V]キーを同時に押して、スクリーンショットを[ペイント]に貼り付ける（または、メニューバーの[編集]より[貼り付け]を選択し貼り付けてください）
・[ペイント]上にはエラーメッセージが出ているウィンドウのみのデータが映し出されているはずです。
　エラーメッセージが出ているウィンドウと背景のデスクトップ全体の両方が写り込んだデータがペイント上に貼り付けられている場合は、一番始めのステップからやり直してください。
　[PrintScrean]キーを押す際、[ALT]キーも同時に押されていることにご注意ください
・メニューバーの[ファイル]をクリックし、[名前を付けて保存]を選択する
・[ファイルの種類]から「JPEG」を選択し、サポート送信用にファイルの大きさを小さくする
・ファイルに分かりやすい名前（例：日付と氏名）を付け、デスクトップ上など見つけやすい場所へ保存する
・終了後、[ペイント]を閉じる
